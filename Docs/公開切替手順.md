# 本番公開 切替手順（noindex 解除 チェックリスト）

作成: 2026-07-22 ／ 対象: P・SPO リラクゼーション（`pspo-relaxation.jp`）

> ⚠️ **この手順は「独自ドメインが Cloudflare Pages に接続され、`https://pspo-relaxation.jp/` で表示できる状態」になってから実行する。**
> ドメイン接続前に noindex を解除すると、`pspo-relaxation.pages.dev` の方がインデックスされ、
> 後から重複コンテンツ・不要なURLの掲載につながる。

- [ ] `pspo-relaxation.jp` のネームサーバー切替が完了し、DNS が Cloudflare に移っている
- [ ] **メール（MX / SPF / DKIM）が切替後も正常に送受信できることを実機確認済み**
- [ ] Cloudflare Pages のカスタムドメインに `pspo-relaxation.jp`（＋ `www`）を追加済み
- [ ] `https://pspo-relaxation.jp/` が正しく表示され、`/menu` 等のクリーンURLも全て開ける
- [ ] プライバシーポリシー（`/privacy`）の **【要確認】プレースホルダが全て実データに置換済み**
      → `grep -n 'class="todo"' mocks/page-privacy.html` が **0件** になること

## Step 1. noindex 3層をすべて解除する

**3層すべてを外さないと解除にならない。1つでも残るとインデックスされない。**

### 層1: 各HTMLの `<meta name="robots">`

```bash
cd mocks
grep -rln 'name="robots" content="noindex' *.html assets/page-template.html
# → 該当行を削除（または content="index, follow" に変更）
```

⚠️ **`viewer.html` / `mock-a-premium.html` / `mock-b-natural.html` / `mock-c-clean.html` /
`mock-d-irodori.html` / `_contact.html` の6ファイルは開発用のため noindex のまま残すこと。**

### 層2: `mocks/robots.txt` を本番版へ差し替え

```
User-agent: *
Allow: /

# 開発用ファイル（恒久的に検索除外）
Disallow: /viewer.html
Disallow: /mock-a-premium.html
Disallow: /mock-b-natural.html
Disallow: /mock-c-clean.html
Disallow: /mock-d-irodori.html
Disallow: /_contact.html

Sitemap: https://pspo-relaxation.jp/sitemap.xml
```

### 層3: `mocks/_headers` の `X-Robots-Tag`

`/*` ブロックの **`X-Robots-Tag: noindex, nofollow, noarchive` の行だけ削除**する。
同ブロックのセキュリティヘッダ（HSTS / X-Frame-Options / Referrer-Policy / Permissions-Policy）は**必ず残す**。
ファイル後半にある開発用ファイル個別の `X-Robots-Tag: noindex, nofollow` も**そのまま残す**。
